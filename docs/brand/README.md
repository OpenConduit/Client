| File | Description |
|---|---|

## Colors
| Name | Hex | Usage |
|---|---|---|
| Electric Blue | #3B82F6 | Primary actions, links |
| Electric Violet | #8B5CF6 | Secondary accents, gradients |
| Cyan | #06B6D4 | Code, badges, tertiary |
| Dark Navy | #0F172A | Primary background |
| Surface | #1E293B | Cards, panels |
| Muted Gray | #64748B | Placeholder, secondary text |
| Off White | #F8FAFC | Body text |

## Fonts
- Headings: **Inter Bold**
- Body: **Inter Regular**
- Code: **JetBrains Mono**

## Logo Rules
- Dark backgrounds only
- Never recolor or stretch
- Min size: 32×32px
- Clear space: 1× icon height on all sides
